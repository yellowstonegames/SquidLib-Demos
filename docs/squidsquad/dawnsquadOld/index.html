Use the mouse to select a path; click to confirm move. You can also move using arrow keys, WASD, or the numpad, and can stay in place with '.' or numpad 5. Move into enemies to destroy them.


This demo uses the DawnLike tileset by DragonDePlatino and DawnBringer. The specific variant used is DawnLikeAtlas, which splits up DawnLike into individual tiles for usage by libGDX in a TextureAtlas. The tileset is CC-BY-4.0 licensed. The code is Apache 2.0 licensed.
